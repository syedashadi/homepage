My Resources

I teach Web Programming, Web Technology, Internet Programming, Structured Query Language (SQL), Human Computer Interaction and Introduction to New Media.

Web Programming

The course is designed to present fundamentals, technologies and components for web application developments.

Web Technology

This course will present the latest technology in producing innovative WEB by WEB prominent example of the success projects.

Int. Programming

This course covers the development of web component with Servlets and JSP technologies using the Apache Tomcat/ GlassFish open source application server.

Structured Query Language (SQL)

SQL is Structured Query Language, which is a computer language for storing, manipulating and retrieving data stored in relational database.

Human Computer Interaction

This course presents the physical and informational aspects of the human computer interaction. It emphasises on the process of user interface design and development including user-centered design and task analysis.

Introduction to New Media

This class teaches the new user how to get their blog up and started quickly while teaching students to fully understand the right way to install, administer, customize and market their blog.
